# UIImageView-MultiAvatar

项目中需要用到即时通讯的功能，在做群聊的时候，群头像仿的是微信的UI

## 使用方法

把 UIImageView+MultiAvatar 这个分类拖到项目中，根据实际需求，修改头像大小，间距等参数，使用的时候类似于 SDWebImage：

```
[avatar ht_setImageWithURLStrArr:imageUrlStrArr];
```
